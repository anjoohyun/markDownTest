# 정규표현식

정규신, Regular Experssion

## 역할

- 문자 검색(Search)
- 문자 대체(replace)
- 문자 추출(extract)

## 테스트 사이트
https://regexr.com

## 정규식 생성

```js
//생성자
new RegExp('표현','옵션')
new RegExp('[a-z]','gi)

//리터럴
/표현/옵션
/[a-z]/gi
```
